# Next.js - yarn

1. 설치

```bash
## 최상위 디렉토리에서
yarn create next-app
```

2. 프로젝트 세팅

```
yarn create ..

  success Installed ..
    - create-next-app
  ✅ What is your project named? ... [프로젝트 이름 작성]
  ✅ Would you like to use the recommended Next.js defaults? › Yes, use recommended defaults
  (app router / esLint / tailwind css / ts)

  Using yarn.
  ...

  Initialized a git repository.
```

3. 실행

```bash
yarn dev
```

4. 로컬 서버 접속

- Local: http://localhost:3000
